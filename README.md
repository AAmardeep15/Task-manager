# Task Manager Web App

A React + Vite task manager that supports adding, completing, and deleting tasks.

## Features

- Add new tasks with input validation (trims whitespace and blocks empty tasks)
- Mark tasks as completed with checkboxes
- Delete tasks
- Task counters for total, pending, and completed
- Persistent storage in `localStorage`
- Responsive and accessible UI with semantic structure

## Tech Stack

- React (JavaScript)
- Vite
- ESLint

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Run lint checks:

```bash
npm run lint
```
